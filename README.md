# AtomQuest Goal Tracking Portal

## Overview

AtomQuest Goal Tracking Portal is a web-based employee performance and goal management system developed for AtomQuest Hackathon 1.0.

The platform enables employees, managers, and administrators to collaboratively manage organizational goals, quarterly progress tracking, and approval workflows through a centralized digital portal.

---

## Features

### Employee Module

* Create and manage goals
* Set targets and weightages
* Submit quarterly progress updates
* Track achievement status

### Manager Module

* Review employee goals
* Approve or reject submissions
* Add review comments and feedback
* Monitor team progress

### Admin Module

* Manage user roles
* View analytics and dashboards
* Monitor overall system activity

---

## Technology Stack

* Frontend: React + Vite
* Backend & Database: Supabase PostgreSQL
* Authentication: Supabase Auth
* Hosting: Render
* Version Control: GitHub

---

## Architecture

Users → React Frontend → Supabase Authentication → PostgreSQL Database

---

## Validation Rules

* Total goal weightage must equal 100%
* Minimum weightage per goal: 10%
* Maximum goals per employee: 8
